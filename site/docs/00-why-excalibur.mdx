Excalibur is designed with approachability in mind. You can make some [classic favorites](/docs/getting-started) with just a few lines of code! Excalibur will meet you where you're at, whether you're new to game development or an experienced game maker.

Excalibur has a ton of features to help you make games!
<ul style={{listStyle: 'none', padding: 0 }}>
    <li>📖 Lots of documentation at https://excaliburjs.com <ul><li>[Samples](/samples) & [Showcase](/showcase)</li></ul></li>
    <li>📦 Objected-Oriented TypeScript first API similar to Unity/Godot</li>
    <li>🎥 Friendly API based around a theater metaphor of [Scenes](/docs/scenes), [Actors](/docs/actors), and [Actions](/docs/category/actions/)</li>
    <li>🏃[SpriteSheet](/docs/spritesheets), [Sprites](/docs/sprites), and [Animations](/docs/animation)</li>
    <li>💥 Built in [collisions](/docs/colliders) with [Arcade](/docs/physics#arcade) and [Realistic](/docs/physics#realistic) Physics</li>
    <li>🧩 [2D](/docs/tilemap) and [Isometric](/docs/isometric) TileMaps</li>
    <li>🎨 [Custom shader](/docs/materials), [post-processing](/docs/postprocessors/) support, and [render plugins](/docs/custom-renderer-plugins)</li>
    <li>🖼️ Auto-batching draw calls and really tuned draw performance</li>
    <li>🎁 [Entity-Component-System](/docs/entity-component-system) under the hood when you need control</li>
    <li>🛠️ Popular Game Dev Tool Support</li>
    * [Tiled maps](/docs/tiled-plugin)
    * [LDtk levels](/docs/ldtk-plugin)
    * [Spritefusion levels](/docs/spritefusion-plugin)
    * [Aseprite image files](https://github.com/excaliburjs/excalibur-aseprite)
    * [JSFXR audio](https://github.com/excaliburjs/sample-jsfxr/)
</ul>

There are a lot of great tools out there to build games on the web and no tool is perfect. Use whichever tool makes you happiest. 😊
